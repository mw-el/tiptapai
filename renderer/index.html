smart_toy Claude Code Integration

Was ist das?

Öffnet ein Terminal mit Claude Code, das deinen aktuellen Text kennt. Claude kann dir beim Überarbeiten von Absätzen helfen.

So funktioniert es

  1. Klicke auf smart_toy um das Terminal zu öffnen
  2. Claude kennt dein Dokument und die Absatz-Nummern (§1, §2, ...)
  3. Gib Claude Anweisungen, z.B.: "Überarbeite §5, mach es kürzer"
  4. Claude zeigt dir den überarbeiteten Text
  5. Sage: "In Zwischenablage" - Claude kopiert nur den Text
  6. Füge mit Ctrl+V im Editor ein

Beispiel-Prompts

  • Zeige §5 - Zeigt den Absatz
  • Formuliere §3 um, professioneller
  • Kürze §12 auf die Hälfte
  • Korrigiere Grammatik in §7
  • Füge nach §10 einen Übergang ein
  • Ergebnis in Zwischenablage - Kopiert den Text

Wichtig

Die Absatz-Nummern findest du in der Datei document-numbered.txt im Kontext-Ordner. Claude liest diese automatisch.
