# Usage Guide: text-polish-basic

## Vorbereitung
1. Markiere den zu ueberarbeitenden Abschnitt.
2. Definiere Ziel: Klarheit, Laenge, Tonfall.

## Ausfuehrung
1. Starte mit dem passenden Prompt-Baustein.
2. Pruefe semantische Treue zum Original.
3. Wiederhole mit engerer Vorgabe, falls Ergebnis zu frei wird.

## Abschluss
1. Waehle eine Variante.
2. Fuehre einen schnellen Lesefluss-Check durch.
3. Uebernimm den Text in den Editor.
