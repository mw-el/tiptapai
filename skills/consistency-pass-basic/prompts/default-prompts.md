# Prompt-Bausteine: consistency-pass-basic

## Inkonsistenzen finden
Pruefe den Text auf inkonsistente Begriffe, Namen und Schreibweisen. Liste nur echte Konflikte.

## Korrekturvorschlag
Erstelle fuer jeden Konflikt einen einheitlichen Zielbegriff und begruende ihn in einem Satz.

## Anwendung
Wende die Korrekturen auf den Text an und gib den bereinigten Text aus.
