# text-polish-basic

## Ziel
Verbessere einen gegebenen Abschnitt sprachlich, ohne Inhalt zu verfälschen.

## Trigger
- Wenn ein Absatz kuerzer, klarer oder professioneller klingen soll.
- Wenn mehrere Varianten mit unterschiedlichem Tonfall benoetigt werden.

## Workflow
1. Lade Prompt-Bausteine aus `prompts/default-prompts.md`.
2. Folge der Vorgehensweise in `references/usage-guide.md`.
3. Nutze bei Bedarf `scripts/run.sh` als Startpunkt fuer Automatisierung.

## Ressourcen
- Prompts: `prompts/default-prompts.md`
- Vorgehensweise: `references/usage-guide.md`
- Skript: `scripts/run.sh`
