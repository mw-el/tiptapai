# gendersmart

## Ziel
Deutschen Text geschlechtsneutral umformulieren: elegant, lesbar, ohne Genderstern oder Binnen-I.
Methode: Genderleicht / EMMA-Prinzip.

## Trigger
- Wenn ein Text auf generisches Maskulinum geprüft werden soll.
- Wenn geschlechtsneutrale Alternativen gesucht werden.
- Wenn der Lesefluss erhalten bleiben soll (kein mechanisches Gendern).

## Workflow
1. Lade den vollständigen Prompt aus `prompts/default-prompts.md`.
2. Folge der Vorgehensweise in `references/usage-guide.md`.

## Ressourcen
- Prompt: `prompts/default-prompts.md`
- Vorgehensweise: `references/usage-guide.md`
