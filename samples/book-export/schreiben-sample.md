# Was ist eine Kurzgeschichte?

::: chaptermeta
part=Teil I — Grundlagen | subtitle=Form, Tradition und das Wesen des Kurzen | intro=Die Kurzgeschichte ist keine verkürzte Version eines Romans. Sie ist eine eigene Form — präziser, konzentrierter, oft brutaler in ihrer Auswahl. Was sie weglässt, ist genauso bedeutsam wie das, was sie zeigt.
:::

Eine Kurzgeschichte beginnt fast immer *in medias res* — mitten in einer Situation, ohne ausführliche Vorgeschichte. Diese Unmittelbarkeit ist kein stilistisches Mittel, sondern eine strukturelle Notwendigkeit: Kurzgeschichten haben nicht die Zeit, sich Zeit zu nehmen.

Entscheidend ist nicht die Wortzahl, sondern die **strukturelle Knappheit**: Eine einzige Szene, ein einziges Problem — und eine Reaktion darauf.

::: pullquote
»Eine Kurzgeschichte erzählt nicht eine Geschichte. Sie erzählt genau eine Geschichte — und streicht alles andere.«
— Flannery O'Connor
:::

## Die Einheit des Augenblicks

Was viele Kurzgeschichten verbindet, ist das Prinzip der **epiphanen Szene**: Ein Moment der Erkenntnis, der Verschiebung, des Verlustes. James Joyce prägte den Begriff der Epiphanie für diese plötzliche Offenbarung im Alltäglichen — und die Kurzgeschichte ist die ideale Form für solche Momente.

Raymond Carver reduzierte dieses Prinzip auf ein Äußerstes. Seine Figuren sprechen wenig, handeln kaum — und dennoch verändern sich die Koordinaten ihrer Welt im Verlauf weniger Seiten grundlegend.

### Abgrenzung zur Novelle

Die Novelle kennt Subplot, mehrere Szenen, eine komplexere Zeitstruktur. Die Kurzgeschichte hingegen ist monofokussiert. Sie kann sich diese Breite nicht leisten.

::: exercise Übung 1.1
Lesen Sie eine beliebige Kurzgeschichte. Markieren Sie den *Moment der Veränderung*: Wann genau kippt etwas? Notieren Sie, was vorher und was nachher anders ist.
:::

## Was eine Kurzgeschichte nicht ist

Eine Kurzgeschichte ist kein Sketch, keine Anekdote, keine Pointe. Das häufigste Missverständnis: Sie sei ein Roman, dem man die meisten Kapitel gestrichen hat. Das Gegenteil ist richtig.
